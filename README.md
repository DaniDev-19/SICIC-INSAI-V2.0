SICIC-INSAI V2.0

Sistema de Información para el Control de Inspecciones de Campo, diseñado para la gestión integral de procesos fitosanitarios y operativos del Instituto Nacional de Salud Agrícola Integral (INSAI).

---

Sobre el Proyecto
Este sistema es la evolución (V2.0) de la plataforma original desarrollada para la automatización de los procesos fitosanitarios. Mientras que la versión anterior sentó las bases para el registro de inspecciones, la V2.0 unifica el Frontend y el Backend en una arquitectura más robusta, escalable y eficiente.

---

Descripción
SICIC-INSAI V2.0 es una plataforma web integral que permite a los usuarios digitalizar y controlar el flujo de trabajo de las inspecciones agrícolas. El sistema facilita la centralización de datos, mejorando la toma de decisiones y la trazabilidad de las actividades de campo.

---

Evolución del Sistema
V1 (Procesos Fitosanitarios): Enfocada en la digitalización inicial de los datos fitosanitarios, sentando las bases del modelo de datos y las inspecciones básicas.

V2.0 (Sistema Integral - SICIC): Mejora la experiencia de usuario, optimiza las consultas al servidor y centraliza el control de inspecciones de campo bajo un flujo de trabajo más dinámico.
